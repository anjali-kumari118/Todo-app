# Todo List App

A simple web-based todo list application using HTML, CSS, and JavaScript.

## 1. Project Description

This project is a lightweight todo list web app that lets you add, remove, and track tasks. All logic runs in the browser using standard web technologies.

## 2. Features

- Add new tasks to the todo list
- Mark tasks as completed
- Remove tasks from the list
- Simple and responsive UI using HTML/CSS

## 3. Project Structure

```
Todo-app/            # project root
├── todo.html        # main HTML page (todo list UI)
├── todo.css         # styles for the todo list UI
├── todo.js          # frontend behavior (add/remove tasks)
├── requirements.txt # optional dependencies (if used)
├── README.md        # this file
```

## 4. Installation Steps

No installation is required if you just want to open the app locally.

1. Open `todo.html` in your web browser.

If you want to keep a Python virtual environment (optional):

1. Open a terminal in the project folder.
2. Create a virtual environment:
   ```bash
   python -m venv venv
   ```
3. Activate the virtual environment:
   - **Windows (PowerShell)**
     ```powershell
     .\venv\Scripts\Activate.ps1
     ```
   - **Windows (cmd)**
     ```cmd
     .\venv\Scripts\activate.bat
     ```
   - **macOS / Linux**
     ```bash
     source venv/bin/activate
     ```
4. (Optional) Install requirements if you add any:
   ```bash
   pip install -r requirements.txt
   ```

## 5. How to Run the Project

1. Open `todo.html` in a web browser.
2. The app will run locally in the browser without any server.

## 6. Technologies Used

- **HTML** for the UI structure
- **CSS** for styling
- **JavaScript** for frontend interactivity
